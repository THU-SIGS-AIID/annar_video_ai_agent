Agent Architecture

How the AI agent thinks and acts autonomously

🎯 User Request

"Save idea & create script"

→

🧠 LLM Reasoning

Analyze intent & plan

→

🔧 Tool Execution

Perform actions

→

✅ Final Answer

Task complete!

The Loop: If tools were used, results are fed back to the LLM for further reasoning. This repeats until the task is complete (max 5 iterations).

Available Tools

7 powerful tools that the agent can use autonomously

save_idea

IDEA

Save video ideas with title, description, tags, and platform.

save_idea("Morning routine", tags=["productivity"], platform="tiktok")

list_ideas

IDEA

Browse and filter saved ideas by platform or tags.

list_ideas(platform="tiktok", tag="lifestyle")

create_script

SCRIPT

Generate AI-powered scripts optimized for specific platforms.

create_script("Coffee tips", platform="instagram", duration="30s")

save_script

SCRIPT

Save generated scripts to drafts or final folders.

save_script("script.txt", content, is_final=True)

create_project

PROJECT

Create organized video project folder structures.

create_project("Summer Vlog", description="Travel content")

organize_video_file

FILE

Move video files to appropriate project folders (raw/edited/final).

organize_video_file("video.mp4", "Summer Vlog", "raw")

list_projects

PROJECT

Display all projects with statistics and status.

list_projects()

Quick Start

Get up and running in under 5 minutes

# 1. Clone the repository
git clone https://github.com/yourusername/video-creator-agent.git
cd video-creator-agent

# 2. Install dependencies
pip install -r requirements.txt

# 3. Set your OpenAI API key
export OPENAI_API_KEY="your-key-here"

# 4. Run the agent!
python agent.py "Save idea: morning routine for TikTok"
