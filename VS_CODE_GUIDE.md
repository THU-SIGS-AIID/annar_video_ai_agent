# 🎯 How to Use Agent in VS Code

## Method 1: Built-in Terminal (Easiest) ⭐

1. **Open terminal in VS Code:**
   - Press `Ctrl + `` (backtick)
   - Or View → Terminal

2. **Run agent commands:**
   ```bash
   python agent.py "List all ideas"
   python agent.py "Create project called Test"
   ```

---

## Method 2: VS Code Tasks

1. **Press `Ctrl + Shift + P`**
2. **Type "Tasks: Run Task"**
3. **Select "🎬 Run Agent"**
4. **Enter your task when prompted**

**Quick Tasks:**
- `Ctrl + Shift + B` → Run Agent (default)
- Or select from task list:
  - 📋 List Ideas
  - 📁 List Projects

---

## Method 3: Debug Panel

1. **Click "Run and Debug" in left sidebar** (or `Ctrl + Shift + D`)
2. **Select "🎬 Run Agent"**
3. **Enter your task**
4. **Press F5 or click Play button**

---

## Method 4: Web Interface (Visual)

1. **Open `web_interface.html` in browser**
2. **Type your task**
3. **Click "🚀 Run Agent"**

⚠️ Note: Web interface shows how it looks, but still needs terminal for actual execution.

---

## 📋 Quick Command Reference

Copy and paste these into terminal:

```bash
# Ideas
python agent.py "List all ideas"
python agent.py "Save idea: cooking tips for Instagram with tags food and lifestyle"

# Scripts
python agent.py "Create 30s script about productivity"
python agent.py "Create script for TikTok about morning routine"

# Projects
python agent.py "Create project called Summer Vlog"
python agent.py "Show all projects"
```

## 🚀 Pro Tips

1. **Keep terminal open** - Faster than reopening
2. **Use Up Arrow** - Recall previous commands
3. **Tab completion** - Auto-complete file names
4. **Split terminal** - View multiple outputs (`Ctrl + Shift + 5`)
